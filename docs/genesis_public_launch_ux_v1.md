# Genesis Public Launch UX Mission v1

**KPI:** обычный человек открывает сайт и через 30 секунд хочет продолжить разговор с Genesis.

## Сделано в этой миссии

1. **API убран с публичного сайта** — Setup Wizard больше не перехватывает `/site`; только `/setup` (Owner Mode, localhost).
2. **Первый экран** — приветствие Genesis, кнопки «Голос» и «Написать сообщение», без технических терминов.
3. **Голос** — проверка разрешения через Permissions API; человеческий текст; 127.0.0.1; обработка no-speech.
4. **Чат без API** — Genesis Brain + Local Mind отвечает всегда.
5. **Discovery** — мягкие подсказки возможностей после тем (кофейня, салон…).
6. **Ошибки** — без «порт 8000», «backend», «LLM».

## Не делать до релиза

Новые AI-модули, COO, Wallet UI, Marketing Lab, Trading, Smart Routing.

## Проверка глазами нового человека

1. Открыл `/site`
2. Прочитал приветствие
3. Написал «Привет»
4. Получил живой ответ
5. Нажал микрофон → понятная инструкция или сразу запись
6. Вернулся — память по `visitor_id`
