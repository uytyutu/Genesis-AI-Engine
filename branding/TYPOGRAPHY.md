# Virtus Core — Typography

## Stack

- **Display:** Segoe UI Variable, SF Pro Display, system-ui, sans-serif
- **Body:** Segoe UI Variable, SF Pro Text, system-ui, sans-serif
- **Mono (dev):** Cascadia Code, SF Mono, Consolas, monospace

## Scale

| Role | Size | Weight | Tracking |
|------|------|--------|----------|
| Brand wordmark | 26–32px | 600–700 | -0.02em |
| Assistant name (Vector) | 24–28px | 700 | 0.04em |
| Signature (by Virtus Core) | 10–11px | 500 | 0.2em |
| Body | 15–16px | 400 | normal |
| Caption | 11–13px | 400–500 | 0.05em |

## Rules

1. Signatures use uppercase tracking (`tracking-[0.2em]` in UI).
2. Never use decorative script or generic “tech” fonts.
3. Wordmark is vector outlines in `virtus-core-logo.svg` — not a web font substitute for the logo.
