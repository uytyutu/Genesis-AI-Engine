## Принципы

Genesis продаёт **не AI и не модели**.

Genesis продаёт (когда доказано рынком):

| Ценность | Пример |
|----------|--------|
| Результат | Готовый сайт, работающий CRM |
| Экономия времени | Меньше ручной рутины |
| Автоматизация | Повторяющиеся процессы |
| Цифровые сотрудники (роли) | Digital Sales Manager |
| Организационная память | Company Brain |
| Рост эффективности | Измеримый ROI |

> **Цена определяется ценностью для клиента** — не количеством токенов и не списком моделей.

*Evidence gate:* все цифры ниже — **гипотезы**. Окончательные цены — после EL3 Deep Review (первый клиент, себестоимость, API, реакция рынка).

## Предварительная стратегия подписок (гипотеза)

> Не публиковать как обещание клиенту до EL3. Диапазоны — для внутреннего планирования.

| Тариф | Ориентир €/мес | Для кого |
|-------|----------------|----------|
| **Free** | 0 | Знакомство. Базовый чат. Жёсткие лимиты. |
| **Personal** | 19–49 | Один человек, фриланс |
| **Pro** | 79–149 | Профессионал |
| **Business** | 249–499 | Малый бизнес |
| **Business+** | 699–1 499 | Компания с несколькими ролями |
| **Enterprise** | **Индивидуально** | См. ниже — **не** фиксировать €1 200 как «стандарт» |

### Enterprise — модель ценообразования

**Публично:** «Свяжитесь с нами» → анализ потребностей → индивидуальное предложение.

**Внутренний ориентир:** от **~2 000 €/мес** — только если ROI для клиента доказан (экономия времени/ФОТ, интеграции, безопасность, поддержка).

Корпоративные AI-платформы часто используют:

- цену за пользователя;
- цену за интеграцию;
- SLA и compliance;
- объём автоматизации.

**Не** единый публичный тариф «€1 200 для всех».

*Value-based pricing (будущее):* если Genesis **доказано** экономит клиенту десятки тысяч €/мес, несколько тысяч € подписки — инвестиция, не «дорогой чат». Только после измерений на реальных клиентах.

## Связь с Digital Employee Strategy

Клиент покупает **роль**, не подписку на ChatGPT:

```
Genesis Core
    + Digital Sales Manager     → €X/мес (когда доказано)
    + Digital Software Engineer → €Y/мес
    + …
```

Подписка Platform + add-on роли из Marketplace — масштабируемее, чем «всё в одном тарифе».

*Mission 1:* клиент покупает **результат** (сайт), не роль по подписке.
