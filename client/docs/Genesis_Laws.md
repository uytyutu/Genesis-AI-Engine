# Genesis Laws

**Статус:** FROZEN — менять только при серьёзном блокере Play Mode.  
**Версия:** 1.1 · 2026-07-04  
**Лимит:** 5 законов. Новые — только если один из пяти не покрывает случай.

Не код. Фундамент, к которому возвращаемся при каждом архитектурном решении.

**Фильтр новой функции:**

> Соответствует ли она **законам Genesis**?  
> Да → развивать. Нет → пересмотреть идею.

---

## Genesis Law №1 — Plan → Approve → Act

Никаких **критичных** действий без подтверждения CEO.

```
Анализ → План → Подтверждение CEO → Выполнение → Отчёт
```

Работает везде:

* продажи (КП, письмо, договор);
* разработка и код;
* deploy и публикации;
* деньги и выплаты;
* изменение важных данных.

Genesis — **цифровой исполнительный аппарат**, не автономный агент.  
Studios готовят ночью → утром CEO получает **список на утверждение**.

---

## Genesis Law №2 — One Window

Genesis — место, где **начинается рабочий день**.

Не заменяя все сервисы сразу, а **объединяя** их в одном рабочем пространстве.

| Сигнал | Вопрос |
|--------|--------|
| Открываю GitHub | Закрывает ли Genesis git-задачу? |
| Открываю Railway / Vercel | Нужен ли статус и deploy внутри? |
| Браузер за проектом | Должны ли данные быть в Desktop? |

---

## Genesis Law №3 — Dogfood First

Любая новая возможность **сначала** используется Genesis Company.

Помогает нам работать → может стать продуктом для клиентов.  
Не помогает нам → зачем рынку?

---

## Genesis Law №4 — Evidence Before Automation

Автоматизировать только то, что **уже доказало ценность вручную**.

Пример (Sales):

1. CEO сам продал N раз / провёл outreach.
2. Понял, какие письма работают.
3. Понял, как назначается цена.
4. **Только тогда** Sales Studio автоматизирует процесс.

Защита от автоматизации неработающих идей. Связь с **уровнями зрелости** (`Genesis_Company_OS_Maturity_v1.md`).

## Genesis Law №5 — Human Accountability

Genesis помогает принимать решения. **Ответственность всегда у человека.**

| Genesis делает | CEO делает |
|----------------|------------|
| Рекомендует цену | Подтверждает |
| Готовит договор | Подписывает / отправляет |
| Предлагает перевод | Утверждает |
| Показывает последствия | Решает |
