# Daily Driver Journal

**Цель:** записывать трение при ежедневном использовании Genesis Desktop.  
**Правило:** идеи из журнала ценнее архитектуры без практики.

**Запуск:** `cd client/desktop && npm run dev`

---

## Как пользоваться

Каждый рабочий день — 2–5 минут в конце:

1. Открывал ли Desktop первым (да/нет)?
2. Что сделал через Desktop vs браузер/Cursor?
3. Что мешало / чего не хватало?
4. Одна вещь, которая сделала бы Desktop «единственным окном».
5. *(опционально)* Сигналы для Business Readiness — что произошло на рынке (продажа, клиент, KYC)?

**Пауза:** философию и score **не усложнять** — копить истории до реальных пользователей.
